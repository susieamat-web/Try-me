Controls
Left/Right: move
Up: jump
Space: shoot
P: pause / resume
Pause button (top bar): pause / resume anytime
R: restart
Gameplay
3 worlds (tile-based levels with gaps, bricks, question blocks, pipes)
Enemies can be stomped or shot
Boss spawns near the end of each world; reduce boss HP by shooting
Score + highscore saved in localStorage
